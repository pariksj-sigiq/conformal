## Design notes

The deck was revised to:

1. Feel closer to the earlier Conformal sales presentation.
2. Add more detail and proof objects on each page.
3. Remove heavy red banner callouts.
4. Use quieter green/neutral notes.
5. Keep the narrative focused on source-system truth, field discovery and concrete asks.

## Narrative spine

1. SFS gave a use-case document.
2. Conformal decomposed it into a 146-field schema hypothesis.
3. The real gap is smaller than "94 unavailable" sounds because 58 fields are computed and 27 are externally sourced.
4. Data access should start with masked samples and schemas.
5. The SAP BW / Power BI route is the first path to validate.
6. By the SAP and field-force working sessions, 84 of 146 fields can be dispositioned.
7. Wave 1 should focus on Enterprise Assistant, Field Performance and Dealer Profiles.
