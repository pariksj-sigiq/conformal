# SFS Client Work Outputs

This folder contains generation scripts and visual QA outputs for the SFS client-facing document and deck.

## Scripts

| File | Use |
|---|---|
| `build_client_artifacts.py` | Generates the client-facing deep-dive document artifacts. |
| `build_detailed_deck.py` | Generates the detailed SFS data deep-dive deck. |

## Rendered outputs

| Folder/file | Use |
|---|---|
| `rendered_doc/` | Page-by-page PNG render of the deep-dive PDF for visual review. |
| `rendered_deck_notes/` | Page-by-page PNG render of the latest deck after the red-banner cleanup. |
| `rendered_deck/`, `rendered_deck_v2/`, `rendered_deck_polished/` | Earlier deck render iterations. |
| `doc_contact_sheet.png` | Quick review sheet for the document. |
| `deck_contact_sheet.png`, `deck_v2_contact_sheet.png`, `deck_polished_contact_sheet.png` | Deck review sheets from successive iterations. |
